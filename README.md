Une entreprise de vente d’équipements informatiques (ordinateur, chargeur, souris, clavier, casque, téléphone, tablette) vous appelle pour analyser leurs données de vente afin de prendre des décisions stratégiques. 
La base de données a été endommagée à cause de la numérisation des données, ces dernières n’étant disponibles que sur les factures versions papiers. Les données mises à votre disposition sont contenues dans le fichier ventes.xlsx.
